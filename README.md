# DOM Manipulation Project

## Overview
This project demonstrates JavaScript DOM (Document Object Model) manipulation techniques. The webpage updates content dynamically and responds to user interactions.

## Features
- Dynamic text updates
- Event handling using buttons
- DOM element selection
- CSS style modification through JavaScript

## DOM Techniques Demonstrated

### 1. Element Selection
```javascript
document.getElementById()
```

### 2. Event Handling

### 3. Content Modification
